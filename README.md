## 📥 Installation

Clone the repository

```bash
git clone https://github.com/swapnika9494/portfolio.git
```

Navigate to the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
